# Self service DB

A Next.js application which allows users to create their own databases in a PostgreSQL速 instance

## Environment variables

| Variable    | Description                                       | Default   |
| ----------- | ------------------------------------------------- | --------- |
| DB_HOST     | PostgreSQL速 host                                  | localhost |
| DB_PORT     | PostgreSQL速 port                                  | 5432      |
| DB_USER     | Administrator username of the PostgreSQL instance | postgres  |
| DB_PASSWORD | PostgreSQL速 administrator password                | password  |
| DB_USE_SSL  | Whether to connect to PostgresQL using SSL or not | false     |
| DB_INSECURE | Whether to allow insecure DB connections          | false     |
